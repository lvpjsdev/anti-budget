# Anti-Budget Deployment Guide

## Live URLs

- **GitHub Pages:** https://lvpjsdev.github.io/anti-budget/
- **Telegram Mini App:** `https://lvpjsdev.github.io/anti-budget/` (set in BotFather)

## Telegram Mini App Setup

### 1. Create Bot
1. Open @BotFather in Telegram
2. `/newbot`
3. Name: "Anti-Budget"
4. Username: `@anti_budget_bot` (or unique)

### 2. Set Menu Button
1. `/mybots` → Select Anti-Budget
2. `Bot Settings` → `Menu Button` → `Configure`
3. URL: `https://lvpjsdev.github.io/anti-budget/`

### 3. Create Mini App (Optional - for full features)
1. @BotFather → `/newbot` → choose "Bot API"
2. Use this URL: `https://lvpjsdev.github.io/anti-budget/`

## Testing
- Open GitHub Pages link in browser
- Or open bot → menu button → Mini App
