# Anti-Budget — Autonomous Execution Rules

## Принцип: Минимум отвлечений. Максимум работы в фоне.

---

## ✅ МОГУ САМ (без тебя)

| Что | Как часто | Тебя |
|-----|-----------|-----|
| Мониторить конкурентов | Каждый день 09:00 | Не беспокоить |
| Писать код (kilo run --auto) | По расписанию | Не беспокоить |
| Коммитить и пушить | После каждого шага | Не беспокоить |
| Обновлять документы | По мере работы | Не беспокоить |
| Собирать данные для Reddit-поста | Фон | Не беспокоить |
| Писать логи и прогресс | Постоянно | Не беспокоить |

## ⚠️ КРИТИЧНОЕ — ЖДАТЬ ТЕБЯ

| Что | Почему |
|-----|--------|

| Публиковать в Telegram | Нужен твой бот-токен |

---

## 📩 КОГДА ПИСАТЬ ТЕБЕ

Только в этих случаях:

1. **Нужно решение** — есть 2+ варианта и без тебя не выбрать
2. **Что-то сломалось** — ошибка, которую не могу пофиксить
3. **Готов MVP** — пора запускать, нужна кнопка
4. **Milestone reached** — "10 users!" или "100 emails!"

---

## 🔕 НЕ ПИСАТЬ

- "Обновление конкурентов готово" — это в лог
- "Код написан" — это в git
- "Коммит сделан" — это в git
- Ежедневные статусы — только в digest в 21:00
